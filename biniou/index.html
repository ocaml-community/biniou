Library biniou

This library exposes the following toplevel modules:

  • Bi_dump
  • Bi_inbuf Input buffer
  • Bi_io Input and output functions for the Biniou serialization format
  • Bi_outbuf Output buffer
  • Bi_share [not for general use]
  • Bi_stream Streaming utilities (experimental)
  • Bi_util Error handling etc.
  • Bi_vint Vint: variable-length representation of integers
